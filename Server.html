Install:

To install open a termina and cd into where FYC is install for a expmale the cd command may look like this 
                        
                            $cd /home/${user}/FYC-Rewrite-V2

Once cd'ed into make sure that you are in the root dir this means where all the .html files are you can cheack by running ls for an expmale 
                                
                                $ls
                                index.html script.js css.css about.html JS CSS Configs...

Now you can run the install command witch is the following command listed bewlow. 
                                        
                                            wget "https://raw.githubusercontent.com/HttpAnimation/FYC-Rewrite-V2/NodeJSSever/InstallWget.sh" && chmod +x InstallWget.sh && ./InstallWget.sh

If the command failed to run you will need npm/node.js to get this do this

1) Fedora if you are using Fedora then you can just run the following command. 

                                                
                                                    sudo dnf install nodejs
2) If you are on anything else head here | https://nodejs.org/en/download
