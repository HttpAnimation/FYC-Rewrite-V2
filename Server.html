Server

The server on FYC allows you to broadcast the FYC website network-wide as a real website.

Install:

To install, open a terminal and navigate to the directory where FYC is installed. For example, the cd command may look like this:

                
                    $ cd /home/${user}/FYC-Rewrite-V2

Once you are in the root directory (where all the .html files are located), you can check by running ls. For example:

                
                    $ ls
                    index.html script.js css.css about.html JS CSS Configs...

Now, you can run the install command. The following command is listed below: 

                
                    $ wget "https://raw.githubusercontent.com/HttpAnimation/FYC-Rewrite-V2/NodeJSSever/InstallWget.sh" && chmod +x InstallWget.sh && ./InstallWget.sh

If the command fails to run, you will need npm/node.js. To install, follow these steps:

  1. If you are using Fedora, run the following command: 

                            
                            $ sudo dnf install nodejs

  2. If you are using any other operating system, visit this link to download npm/node.js: https://nodejs.org/en/download
